  1. http-kit http://http-kit.org

    http-kit is a minimalist, event-driven, high-performance Clojure HTTP server/client library with WebSocket and asynchronous support. Features High performance， High concurrency（like Nginx）， Simple and fridenly API。The implementation is quite simple (about 3200 lines of code). It got some attentions in Clojure communitiy, and was voted in the front page of Hacks News for 2 times. It also received about 400 stars in GitHub.on github

    IO and threading model：Single thread（event loop）read socket，decode it as request，A thread pooling to compute request=>response, encode response，try to write the response to client， If buffer is full, then queue the unwritten respons to the event loop thread, wait for the buffer get flushed.

    Comparing to single thread based Node.js, http-kit is multi-threaded, so doing blocking database query is just fine.
